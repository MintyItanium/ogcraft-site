# OGCraft has officially reached a new milestone.,

I apologise for the text wall you're about to read, I've tried to shorten this but I can't....


As of May 30th 2026, we are now the #1 survival‑only server in the world for the style of gameplay we offer, and the 3rd largest survival‑only server globally, as per FindMC the official Minecraft server listing site.


The two servers above us operate in different categories (one pure vanilla, one modded terrain), which makes OGCraft the largest structured, largest player supported, plugin‑enhanced survival‑only server worldwide.

This achievement belongs to every staff member and every player who has been part of this journey.

### Collin and Caleb  
We started this server almost six years ago, and without you both, OGCraft simply wouldn’t exist. We’ve had our bumps along the way (Map 1 says hello), but we always bounced back. I know I wouldn’t have been able to do that without you.
